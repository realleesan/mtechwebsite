CÁCH UPLOAD:
------------

PHƯƠNG PHÁP 1: FTP (FileZilla, WinSCP)
1. Kết nối FTP đến hosting: ftp.truongvinalogistics.com.vn
2. Navigate đến thư mục public_html
3. Upload các files mới vào đúng thư mục
4. Ghi đè (overwrite) các files đã cập nhật

PHƯƠNG PHÁP 2: cPanel File Manager
1. Đăng nhập cPanel
2. Mở File Manager → public_html
3. Upload từng file vào đúng thư mục

1. Đăng nhập PHPMyAdmin (qua cPanel)
